# SAMP iOS 🎮

GTA San Andreas Multiplayer para iPhone — sem jailbreak!

## Como funciona

Este projeto injeta uma `.dylib` no IPA do GTA SA iOS, adicionando:

- ✅ Menu de servidores SA-MP
- ✅ Chat de texto na tela
- ✅ Conexão com servidores SA-MP 0.3.7
- ✅ Compatível com jogadores Android e PC
- ✅ Sem jailbreak necessário

## Instalação

1. Baixar a `samp_ios.dylib` nos Releases
2. Injetar no IPA do GTA SA usando Sideloadly
3. Instalar no iPhone
4. Abrir o jogo e conectar no servidor

## Compatibilidade

- iOS 13.0+
- iPhone 7 ou superior
- GTA SA versão 2.00+
